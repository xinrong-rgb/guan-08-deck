# 觀無量壽佛經 · 第八觀 像想觀

> 是心作佛 · 是心是佛

Reveal.js 互動簡報，含即時文字雲、單選投票、講師白板。

## 連結

- **學員版（一般觀看）**：https://xinrong-rgb.github.io/guan-08-deck/

## 互動元件

- 第 2 頁 緣起三問（為何要想／想什麼／怎麼想）
- 第 4 頁 法界身遍入心想中
- 第 8 頁 全彩海報全版顯示
- 第 10 頁 一佛二菩薩三蓮華座視覺
- 第 13 頁 出定入定 五元素並列
- 第 14 頁 正觀邪觀對照
- 第 15 頁 即時投票：自我反思念佛偏向
- 第 18 頁 即時文字雲：你帶走的一句話
